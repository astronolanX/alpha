## Overview

This is an example project page. Each project lives in its own folder under `src/content/projects/` with an `index.md` file for content.

## Media

Images and other media for this project should be placed in the `media/` folder within this project directory. You can reference them like this:

```md
![Alt text](./media/screenshot.png)
```

## Features

- Markdown content with frontmatter
- Co-located media assets
- Type-safe schema validation
- Automatic routing
